# BrandMart Electronics (E-Commerce)

Live demo: https://brandmart-electronics.onrender.com

Quick start

- Install dependencies: `npm ci`
- Build: `npm run build`
- Preview a production build: `npx serve dist`

Deploy notes

- This project uses Vite. On Render set the build command to `npm ci && npm run build` and the publish directory to `dist`.
